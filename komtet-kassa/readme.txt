<h3>Инструкция по настройке плагина</h3>

<p>
  Для того, чтобы настроить прием платежей и печать чеков:
</p>

<ul>
  <li>Зарегистрируйтесь на сервисе <a href="https://kassa.komtet.ru/" target="_blank">КОМТЕТ Касса</a>;</li>
  <li>Выполните настройку с выбранным <a href="https://kassa.komtet.ru/integration/software" target="_blank">решением</a>;</li>
  <li>Получите параметры <b>ID магазина</b>, <b>Секретный ключ магазина</b>, <b>ID очереди</b></a>;</li>
  <li>Скопируйте полученные значения в форму настроек плагина.</li>
</ul>

<p>
  Чек возврата формируется при изменении статуса заказа на <b>"Отменен"</b>
</p>

<p>
  Для работы с маркированными товарами необходимо:
</p>
<ul>
  <li>
    В настройках плагина укажите для чека предоплаты - <b>"Оплачен"</b>,
    для чека полного расчёта - <b>"Не выдавать"</b>;
  </li>

  <li>
    Далее в личном кабинете <b>КОМТЕТ Кассы</b> создайте копию чека предоплаты.  Для этого из истории чеков
    необходимо выбрать необходимый чек со способом расчёта "Предоплата 100%", перейти в просмотр чека и нажать "Выдать закрывающий чек";
  </li>

  <li>
    Для каждой позиции нажмите на ярлык справа и добавьте код маркировки с упаковки.
  </li>
</ul>
